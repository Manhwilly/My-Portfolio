ScholarChat AI (RAG System)

An end-to-end Retrieval-Augmented Generation system for academic research

The Challenge

Academic researchers waste significant time sifting through dense papers to find specific, reliable information.

My Solution

Engineered an end-to-end RAG system that allows users to "chat" with a knowledge base of academic papers, providing accurate, sourced answers to complex questions.

Tech Stack

Python, Generative AI, RAG, Vector Databases, Flask API

The Outcome

The system delivers precise, context-aware answers with source citations, dramatically reducing research time and improving the accuracy of information gathering.
